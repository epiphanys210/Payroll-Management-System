# Payroll Management System

A desktop payroll management application built with Python.

## Features

- Employee Payroll Calculation
- Overtime Pay
- Federal, State, Social Security & Medicare Taxes
- SQLite Database
- Payroll History
- PDF Pay Stub Generation
- Export Payroll to CSV
- Tkinter Desktop GUI

## Technologies

- Python
- Tkinter
- SQLite
- ReportLab
