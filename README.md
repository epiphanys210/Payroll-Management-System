# Payroll Management System

A desktop Payroll Management System developed in Python using Tkinter and SQLite.

## Features

- Employee payroll calculation
- Overtime pay calculation
- Federal, State, Social Security, and Medicare tax deductions
- SQLite database integration
- Payroll history viewer
- PDF pay stub generation
- CSV payroll export
- User-friendly Tkinter GUI

## Technologies Used

- Python
- Tkinter
- SQLite
- ReportLab
- CSV

## How to Run

1. Clone this repository.
2. Install ReportLab:

```bash
pip install reportlab
```

3. Run:

```bash
python3 payroll_gui.py
```
